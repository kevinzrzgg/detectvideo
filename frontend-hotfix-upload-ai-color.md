## Risks / notes

- The native file input remains in the DOM for accessibility and label-triggered selection; it is visually clipped to 1px with opacity 0, so browser-localized native labels are not visible.
- Real media processing remains disabled by design; this hotfix only changes the static frontend visuals.
- The repository had unrelated dirty/untracked files before this task (`launch-deploy-report.md`, `scripts/check.mjs`, and several docs). This task only stages the changed source/dist/report files listed above.
