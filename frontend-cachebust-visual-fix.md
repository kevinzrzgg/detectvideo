## Risks / notes

- Browser accessibility trees may still expose the hidden native file input role as a file-upload button with localized fallback text. The visual requirement is satisfied by clipping the native control to 1px with opacity 0 and by displaying only custom English UI.
- Cloudflare normal URLs briefly served the prior HTML immediately after deployment; after revalidation they served the hashed-asset HTML. The new `Cache-Control: no-cache, must-revalidate` header is intended to reduce this stale-HTML risk for future updates.
- Real media processing remains disabled by design; this task only changes the static frontend visuals and cache behavior.
- The repository had unrelated dirty/untracked files before this task (`launch-deploy-report.md`, `scripts/check.mjs`, and several docs). This task stages only the changed source/dist/report files listed above.
